# Daily_News_Summarizer_AI

A Python-based automated assistant that curates, filters, and summarizes daily tech updates (Databricks, Spark, AI/LLMs) using the Gemini API, delivering a clean digest straight to your phone via Twilio WhatsApp.

## Overview

Daily_News_Summarizer_AI is a personal data-engineering and AI assistant designed to eliminate doom-scrolling and information overload. The automation runs daily to fetch, filter, and deliver a tailored tech-news digest directly to WhatsApp.

### Key Features
* **Targeted Scraping:** Tracks specific technical domains (Databricks, Apache Spark, Data Engineering, AI/LLMs) and top content creators.
* **AI-Powered Filtering:** Leverages the Gemini API to analyze source content, filtering out the noise to extract only high-value updates from the last 24 hours.
* **Automated Delivery:** Formats the curated insights into a clean, readable digest and dispatches it via Twilio's WhatsApp API.
